# Layer 3 Roadmap

Phase 1: Instrument
- add event tracking for word attempts, completions, failures, hints, and timing
- persist session summaries

Phase 2: Model
- create per-word mastery
- create per-player learner profile
- create review due dates

Phase 3: Adapt
- select target words from mastery data
- change difficulty dynamically
- adjust hints and pacing based on behavior

Phase 4: Personalize
- show progress by weakness area
- generate daily review sets
- surface phonics patterns and word families

Phase 5: Optimize
- tune the feedback loop
- remove friction
- keep the game fun first

Definition of done:
The game should change behavior because the player changed behavior.
