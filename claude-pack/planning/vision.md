# Vision

The product should feel like a premium spelling game on the surface and an adaptive tutor underneath.

Layer 1:
- play the game
- complete words
- score points

Layer 2:
- levels
- streaks
- rewards
- modes
- unlocks

Layer 3:
- the game learns what the player knows
- the game learns what the player misses
- the game learns when to review
- the game learns how to pace the next challenge

North star:
A returning player should see a different experience because the system has learned from them.
