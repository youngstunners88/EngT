# Mastery Model

Each word should have a mastery score.

Suggested bands:
- 0-20: unknown
- 21-40: recognized
- 41-60: developing
- 61-80: competent
- 81-100: mastered

Signals that increase mastery:
- correct completion
- faster completion
- fewer hints
- repeated success over time

Signals that decrease mastery:
- failure
- timeout
- repeated mistakes
- heavy hint dependency

Important:
Mastery should decay slowly so the game keeps mastered words in review.
