# Curriculum Context

Purpose:
Define how the game teaches vocabulary.

Curriculum goals:
- improve retention
- improve recall speed
- improve word recognition
- improve spelling confidence

Use this folder when deciding:
- which words to teach first
- how words should be grouped
- when a word should come back for review
- how to measure mastery

Rule:
The curriculum should adapt to the learner instead of forcing every learner through the same list.
