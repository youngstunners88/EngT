# Adaptation Engine

The adaptation engine converts telemetry into future decisions.

Rules:
- if the player struggles, simplify
- if the player succeeds quickly, raise challenge
- if a word is missed repeatedly, schedule it for review
- if hints are overused, offer more guidance
- if the session is going well, keep momentum

Outputs:
- next target word
- next session recommendation
- review queue
- difficulty shift
- feedback style

The engine should be predictable enough to feel fair.
