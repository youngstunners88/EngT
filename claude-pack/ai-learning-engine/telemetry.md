# Telemetry

Track at least:
- target word shown
- time to complete
- failed attempts
- wrong letters
- hint usage
- voice usage
- completion count
- streak count
- game over cause
- session length
- mastery change

Telemetry must be useful enough to support:
- difficulty tuning
- learning insights
- session summaries
- future personalization

Rule:
If the event will not inform a decision, do not collect it.
