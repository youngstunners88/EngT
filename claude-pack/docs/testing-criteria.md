# Testing Criteria

Baseline tests to protect the experience:
- game starts from menu
- desktop controls work
- mobile controls work
- words can still be completed
- audio toggles work
- voice fallback works when ElevenLabs is unavailable
- game resets cleanly
- no secret is exposed to the client

Layer 3 tests to add later:
- mastery updates correctly
- review items return on schedule
- weak words are selected more often
- session memory survives reloads if persistence is enabled
